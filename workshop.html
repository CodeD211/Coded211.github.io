Palatine Website Workshop

@ 10:15AM

Beginner: Static websites

Host: Jacob Kim and Tanish Mittal

This beginner-level session comprehensively introduces building static websites using fundamental JavaScript and HTML concepts. Participants will learn the basics of structuring web pages, incorporating JavaScript for interactivity, and understanding essential HTML elements. By the end of this workshop, attendees will have a solid foundation to create their own simple static web pages.

Software Needed:
  • VS Code

Advance: Dynamic Websites

Host: Isaac Ruiz and Norbert Zych

In this advanced workshop, participants will advance their skills in web development by learning to build dynamic websites using JavaScript. The session will cover topics such as event handling, asynchronous programming, and integrating dynamic content with HTML markup. Through hands-on projects, attendees will gain proficiency in creating interactive web applications capable of responding to user input and data changes in real time.

Software Needed:
  • VS Code
  • Node.js

Fremd ML & AI Workshop

@ 11:15AM

Beginner: Introduction to Machine Learning using Kaggle

Host: Jason Zhou

Learn the basics of Machine Learning and what it’s all about using Python and a step-by-step tutorial to create a model that predicts which passengers survived the wreck of the Titanic. Attendees will learn how to use basic modules to analyze tabular data.

Software Needed:
  • Kaggle

Advance: Use computer vision to draw like an artist

Host: Ryan Barretto

Build generative adversarial networks (GANs) from scratch to mimic the artistic styles of Picasso, Van Gogh, and others to create beautiful, never-seen-before images. You will learn how to build complex neural networks from scratch and train them using sophisticated optimization strategies.

Software Needed:
  • Private google account

Schaumburg Databases & APIs Workshops

@ 2:00PM

Beginner: Reading & Writing in Firebase

Host: Paritosh Vaghasiya, Ishaan Patel and Vedant Patil

This beginner-friendly workshop introduces Google’s Firebase datastores and their API. We’ll teach you how to set up an application with Firebase and how to access, upload, and update data in Firebase’s real-time database. We’ll also show you how to create a GitHub repository and publish it to the internet via a GitHub page! Resources will be provided at the end of the workshop if you want to learn more about Firebase and its API.

Software Needed:
  • Any type of computer
  • A personal email that you’ll have access to on Hackathon day
  • A premade GitHub account.
  • Any code editor of your choice (We reccomend VS code)

Advance: Global Chat in Firebase

Host: Dawid Wasniowski and Akshat Shah

This intermediate/advanced workshop covers topics mentioned in the “Reading & Writing in Firebase” workshop while building a functional, slightly more complex application. We will program a global chat room (HTML and CSS will be provided) together while explaining what each part of the code does along the way. Some concepts covered include pushing data, loading child data, and more. We’ll also show you how to create a GitHub repository and publish the final app to the internet via a GitHub page! Resources will be provided at the end of the workshop if you want to learn more about Firebase and its API.

Software Needed:
  • Any type of computer
  • A personal email that you’ll have access to on Hackathon day
  • A premade GitHub account.
  • Any code editor of your choice (We reccomend VS code)
